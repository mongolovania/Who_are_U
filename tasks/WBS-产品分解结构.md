## 一、商业模式：月付订阅制

### 定价策略

```
┌─────────────────────────────────────────────────────┐
│                  你谁啊 月付会员                       │
│                                                      │
│  💰 定价：¥19/月（推荐方案 A）                          │
│  🎁 免费试用：3 天全功能试用                            │
│                                                      │
│  订阅后解锁：                                         │
│  ├── 🧠 无限次决策对话 + 完整决策报告                    │
│  ├── 🏔️ 完整我之山可视化（全部节点、全山视野）            │
│  ├── 💬 生动陪伴记忆（对话历史永久保留、情绪回顾）        │
│  ├── 🏅 隐藏称号系统（成就解锁稀有称号）                  │
│  └── 🎨 外观装扮（独影换装、装饰品商店）                  │
└─────────────────────────────────────────────────────┘
```

### 与旧模式对比

| 维度 | 旧模式（多层计费） | 新模式（月付订阅） |
|------|-------------------|-------------------|
| 下载 | 6元一次性 | **免费下载** |
| 定价 | 19.9单次 / 99次卡 / 99年费 | **单一月费 ¥19** |
| IAP 类型 | 消耗型 + 订阅型混用 | **仅自动续期订阅** |
| 用户决策成本 | 每次对话前需考虑是否购买 | **订阅后畅享** |
| 收入模型 | 单次交易为主 | **持续复购** |
| 免费体验 | 3次试用 | **3天全功能试用** |

## 二、交付策略：两阶段 MVP

> 用户确认架构方案后，采用**两阶段交付**降低风险：

```
MVP v1 (12w)                    MVP v2 (+10w)                    Phase 2+
本地优先·单设备·核心闭环        账户+同步+合规+聚合+智能         增值扩展
─────────────────────────────────────────────────────────────────────→
Week 1──────────12           Week 13─────────────22           Week 23+
                                                                       
✅ 10题测试→对话→报告         ✅ 账户系统+E2E同步             称号/外观/记忆
✅ 我之山+迷雾+节点           ✅ 全球合规(4区域)              独影动画/副峰
✅ 月付订阅+付费墙            ✅ 匿名聚合引擎(称号%)           Android适配
✅ 基础称号+基础外观          ✅ Ombre Brain深度集成          运营后台
```

### 为什么分两个阶段？

| 理由 | 说明 |
|------|------|
| **降低风险** | 账户/同步/合规是全新基础设施，独立验证后再集成 |
| **快速验证** | 先做本地版验证核心价值假设（用户愿意为陪伴+决策付费吗？） |
| **技术解耦** | 本地加密存储 → E2E 同步是无缝升级路径，不破坏现有数据 |
| **法规时间** | 隐私政策审核、多区域部署需要时间，不阻塞核心开发 |

## 三、功能分层（按订阅视角）

### 免费层（下载即可用）

```
免费层：让用户感受到价值，产生订阅欲望
├── 10题初始测试 → 3特质泡泡画像
├── 2次完整对话体验（含决策报告）
├── 我之山基础视图（仅显示免费期节点，迷雾覆盖大部分）
└── 3款基础独影外观
```

### 月付会员层（订阅解锁）

```
会员层：持续价值交付，支撑续费
├── 🧠 无限决策对话
│   ├── 5阶段引导对话引擎（不限制次数）
│   ├── 完整决策报告（含历史回顾）
│   ├── 对话历史永久保留（MVP v1 本地加密 → v2 E2E 同步）
│   ├── 情绪标签自动标记（EmotionTag）
│   └── Ombre Brain 记忆引擎（v2 集成：breath→hold→dream 循环）
│
├── 🏔️ 完整我之山
│   ├── 全山可视化（迷雾散去，完整山脉视野）
│   ├── 所有对话自动生成节点
│   ├── 困难节点纪念碑（独影动作定格）
│   ├── 副峰系统（自定义小目标追踪）
│   └── 年度山脉回顾（成长轨迹总览）
│
├── 💬 生动陪伴与持久记忆
│   ├── 对话情感化设计（温暖语气、适时沉默、个性化回应）
│   ├── 情绪曲线回顾（每周/月度情绪变化可视化）
│   ├── 独影状态变化（随使用深度逐渐清晰）
│   └── 时光轴视图（按时间排列所有节点与故事）
│
├── 🏅 隐藏称号系统
│   ├── 基础成就（首次决策、十次对话…解锁普通称号）
│   ├── 隐藏称号（特殊条件触发，如"深夜倾诉者""百难克星"…）
│   ├── 稀有称号（v2 聚合引擎驱动：时间窗口+绝对稀有度）
│   └── 称号展示（在独影旁显示当前佩戴称号）
│
└── 🎨 外观装扮
    ├── 独影外观商店（帽子、眼镜、斗篷、配饰…）
    ├── 限定外观（v2 差分隐私触发：节日限定、稀有度解锁）
    ├── 外观组合搭配（layer 叠加系统）
    └── 独影换装在山上实时预览
```

## 八、关键设计决策记录

| 决策 | 选择 | 理由 | 日期 |
|------|------|------|------|
| 商业模式 | 月付订阅 ¥19/月 + 3天试用 | 持续收入 + 低决策门槛 | 2026-06-06 |
| IAP 类型 | 仅自动续期订阅 | 简化实现 + Apple 推荐 | 2026-06-06 |
| 交付策略 | 两阶段 MVP（v1 本地 → v2 账户+智能） | 降低风险 + 快速验证 | 2026-06-07 |
| 认证方式 | Email + Apple Sign In | 全球覆盖 + iOS 合规 | 2026-06-07 |
| 数据加密 | E2E (AES-256-GCM + Curve25519) | 服务端零明文 | 2026-06-07 |
| 密钥派生 | Argon2id (password → auth+encryption) | 抗暴力破解 + 服务端无密码 | 2026-06-07 |
| 聚合模型 | 差分隐私 (Laplace noise, ε=0.5) | 保护个体 + 统计可用 | 2026-06-07 |
| 数据驻留 | 4 区域 (EU/CN/US/SG) | 满足 GDPR/PIPL/CCPA/PDPA | 2026-06-07 |
| 记忆引擎 | Ombre Brain 定制版 | 已有成熟情感记忆模型 | 2026-06-07 |
| 无社交稀缺性 | 时间/概率/路径/深度/区域 5 轴 + 差分隐私 | 无需社交资格 + 合规 | 2026-06-06 |
| 匿名用户 ID | 不实现 | 单设备本地存储，设备即用户 | 2026-06-06 |
