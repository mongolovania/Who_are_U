## 一、设计系统

### 1.1 设计理念

```
异世界山脉中的温暖陪伴空间。

三个关键词：
- 温暖 (Warm)     — 不冷峻、不科技感、像篝火旁聊天
- 深邃 (Deep)     — 山是有层次的、记忆是有厚度的
- 专注 (Focused)  — 按钮极少、空间即导航、一个画面只做一件事
```

### 2.2 山体渲染

#### 视觉层次（从山脚到山顶）

| 层 | 高度范围 | 颜色渐变 | 节点密度 | 可见性 |
|----|---------|---------|---------|--------|
| **迷雾层** (Foothills) | 0%–30% | `#8B7E74 → #7B6B8A` | 稀疏（3-5节点） | 免费：迷雾覆盖·会员：清晰 |
| **暖光层** (Mid-Mountain) | 30%–60% | `#7B6B8A → #4A4A6A` | 中等（5-10节点） | 免费：部分可见·会员：完全可见 |
| **星空层** (Peak) | 60%–100% | `#4A4A6A → #1A1A2E` | 密集（10+节点） | 免费：不可见·会员：完全可见 |

```dart
  // 使用 Perlin 噪声生成自然山形
  static const int gridSize = 32;

  // height ∈ [0.0, 1.0] → lerp(foothillColor, midColor, peakColor)

```

#### 动态变化

山体外观随用户 DDI（数据密度指数）变化：

| DDI 阶段 | 山体变化 |
|----------|---------|
| COLD (< 50 数据点) | 山体较矮·颜色偏暗·节点稀疏 |
| WARM (50-200) | 山体升高·暖光层开始显现·节点增多 |
| HOT (200-500) | 山体完整·星空层明显·节点密集·金色纹理出现 |
| RICH (> 500) | 山体雄伟·星空璀璨·金色纹理遍布·偶尔流星粒子 |

> DDI 阶段由 Memory Palace 的 L0 DDA 自适应层提供，山体根据用户当前阶段动态渐变——不是一次性跳跃，而是随着每次对话慢慢变化。

### 2.3 迷雾层

#### 效果规格

```
FogLayer:
  类型: 半透明渐变叠加层
  颜色: mist-gray (#C4C4D0)
  覆盖范围: 山体高度的 0%-70%（免费用户）/ 0%-10%（会员）
  透明度: 
    - dense:   opacity 0.85 (免费用户默认)
    - partial: opacity 0.50 (有过几次对话后)
    - light:   opacity 0.20 (会员)
    - none:    opacity 0.00 (深度会员·RICH阶段)
  动画: 订阅后 3 秒缓慢散去（easeOutQuint）
```

#### 迷雾实现

```dart
class FogLayer extends StatelessWidget {
  final FogLevel level;       // dense | partial | light | none
  final double mountainHeight;

  @override
  Widget build(BuildContext context) {
    return AnimatedOpacity(
      opacity: level.opacity,
      duration: const Duration(milliseconds: 3000),
      curve: Curves.easeOutQuint,
      child: CustomPaint(
        painter: FogPainter(
          mountainHeight: mountainHeight,
          fogLevel: level,
        ),
      ),
    );
  }
}

class FogPainter extends CustomPainter {
  // 使用径向渐变 + Perlin 噪声纹理
  // 产生有机·不规则的迷雾效果
  // 不是纯色矩形覆盖，而是有层次的不均匀迷雾
}
```

### 2.5 独影剪影

#### 状态定义

| 状态 | 视觉表现 | DDI 阈值 | 说明 |
|------|---------|---------|------|
| `faint` | 模糊轮廓·低透明度·几乎看不清 | COLD (< 50) | 新用户·数据极少 |
| `emerging` | 轮廓渐清晰·半透明·开始有辨识度 | WARM (50-200) | 有了一些对话 |
| `clear` | 清晰轮廓·完整剪影·姿态固定 | HOT (200-500) | 深度用户 |
| `vivid` | 生动剪影·微光边缘·随情绪微动 | RICH (> 500) | 超深度用户 |

#### 剪影绘制

```dart
class ShadowFigurePainter extends CustomPainter {
  // 绘制陪伴角色的剪影：
  // - 不具象（不是特定人物/动物）
  // - 轮廓柔和·无锐利边缘
  // - 边缘有暖光光晕（warm-gold @ 15% opacity）
  // - 剪影内部有微弱渐变（山体颜色映射）
  // - 随 DDI 阶段动态提高分辨率

  // 关键：剪影形态会随用户主要特质微调
  // - 冒险家 → 略微前倾·有动势
  // - 守护者 → 直立·稳重
  // - 点子王 → 头部微扬·思考状
  // - 共情者 → 微弯·倾听姿态
}
```

### 2.6 陪伴角色/造物

#### 设计原则

- **不是宠物**（不是猫/狗/龙等）——避免「养成游戏」感
- **是「造物」**——山中自然生成的灵性存在，形态随用户成长而变化
- **可交互但不冒犯**——点击进入对话，但不会主动跳出来打断

#### 视觉形态（待定方向）

| 方向 | 描述 | 优势 | 风险 |
|------|------|------|------|
| A: 光灵体 | 柔和光球·发光粒子·形态随情绪变化 | 极简·不具象·包容性强 | 可能过于抽象 |
| B: 矿物精灵 | 山体矿物构成的小型精灵·有纹理 | 与山主题一致·有质感 | 需精细3D建模 |
| C: 气态生物 | 云雾+星光构成·半透明 | 梦幻·适配世界观 | 渲染复杂 |
| D: 抽象符号 | 不断变化的几何/有机形态 | 极简·现代 | 缺乏情感连接 |

> **当前倾向：A + C 混合** — 以光灵体为基础形态，表面覆盖云雾纹理。随DDI阶段逐渐从模糊光团 → 清晰形态。形态微小变化反映用户主要特质。

#### 交互行为

```
山上状态（未点击）:
  - 在独影剪影旁边自由漂浮/行走
  - 偶尔发出微小粒子光效（每 30-60 秒一次）
  - 随用户对话频率调整活跃度

点击 → 对话转场:
  1. 造物发光 + 放大 (300ms)
  2. 山体缩小 + 模糊 (300ms)
  3. 造物升至画面中央 (500ms)
  4. 对话界面淡入 (300ms)
  总时长 ~1100ms
```

### 2.8 3D 渲染方案选型

#### 四方案对比

| 维度 | CustomPainter 伪3D | Flutter Scene (Impeller) | Unity 嵌入 | Rive 动画 |
|------|-------------------|-------------------------|-----------|----------|
| **3D 真实感** | ⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐ |
| **性能 (60fps)** | ⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| **包体积影响** | 0 | +2-5MB | +20-40MB | +1-3MB |
| **开发复杂度** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐⭐ |
| **社区成熟度** | 纯Flutter·稳定 | 较新·快速迭代 | 成熟 | 成熟 |
| **节点交互** | 需手写射线检测 | 内置 | 内置 | 有限 |
| **动画能力** | 手写 | 内置 | 强 | 强（2D为主） |
| **iOS 审核风险** | 无 | 低 | 中（游戏引擎） | 无 |

#### 推荐方案：分阶段

```
MVP v1 (Sprint 4):  CustomPainter 伪3D
  - 山体: 2.5D 等距投影 + 多层视差滚动
  - 节点: 坐标映射 + HitTest
  - 剪影: CustomPainter 路径绘制
  - 迷雾: Canvas 渐变叠加
  - 风险: 最低·最快验证

MVP v2 → Phase 2+:  评估迁移到 Flutter Scene
  - 如果 CustomPainter 性能达标且用户满意 → 继续优化
  - 如果需要更真实的3D效果 → 迁移到 Flutter Scene
  - Unity 仅作为最后选项（包体积 + 审核风险）
```

#### MVP v1 CustomPainter 伪3D 方案细节

```
等距投影:
  - 固定视角：偏航 30° · 俯仰 45°
  - 山体由多层 Canvas.drawPath 叠加
  - 每层 Y 偏移制造高度感
  - 3 层颜色渐变用 Canvas.drawShader (LinearGradient)

视差滚动:
  - 旋转手势 → 各层以不同速率位移
  - 前景层（节点·剪影）移动 1.0×
  - 中景层（山体纹理）移动 0.6×
  - 远景层（星空背景）移动 0.2×
  - 制造 3D 深度错觉

节点点击检测:
  - 屏幕坐标 → 逆投影 → 山体坐标
  - 检测点击是否在节点 bounding box 内
  - 如果多个节点重叠 → 相机推近后再选择
```

```
  300-800ms 陪伴角色放大至填充屏幕 60%
           | 山体进一步缩小至背景
           | 星空背景暗化
           |
  800-1100ms 对话界面从角色背后淡入
           | 角色缩小归位至顶栏/气泡旁
           | 消息列表出现
           |
  1100ms   完成·对话界面就绪
```

### 5.3 对话 → 我之山（详细帧分解）

```
时间轴:
  0ms      场景: 对话页面
           |
  0-200ms  对话界面淡出 (opacity: 1.0→0.0)
           | 陪伴角色保持
           |
  200-700ms 角色缩小 → 回归山体位置
           | 山体从背景逐渐清晰
           | 星空背景亮起
           |
  700-1000ms 山体恢复原始大小
           | 迷雾/节点/剪影渐显
           |
  1000ms   完成·我之山就绪
```

### 5.4 转场实现

```dart
// Mountain → Chat 转场
class MountainToChatTransition extends CustomTransitionPage {
  // 使用 PageRouteBuilder + 自定义 TransitionBuilder
  // 或 GoRouter 的 CustomTransitionPage

  Widget buildTransitions(
    BuildContext context,
    Animation<double> animation,
    Animation<double> secondaryAnimation,
    Widget child,
  ) {
    return Stack(
      children: [
        // 底层: 缩小的山体（带模糊）
        _buildShrinkingMountain(animation),

        // 中层: 放大的陪伴角色
        _buildZoomingCompanion(animation),

        // 上层: 淡入的对话界面
        FadeTransition(
          opacity: Tween(begin: 0.0, end: 1.0).animate(
            CurvedAnimation(
              parent: animation,
              curve: Interval(0.7, 1.0, curve: Curves.easeOut),
            ),
          ),
          child: child,
        ),
      ],
    );
  }
}
```
